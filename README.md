# Summ
SW Companion Pro

«The Ultimate Summoners War Companion»

Descrição

O SW Companion Pro é uma aplicação desenvolvida para ajudar jogadores de Summoners War a gerir a sua conta de forma inteligente.

O objetivo é criar uma aplicação capaz de analisar automaticamente monstros, runas, artefactos e equipas, utilizando Inteligência Artificial para recomendar melhorias e otimizar toda a conta.

---

Funcionalidades

- Gestão completa de monstros
- Gestão de runas
- Gestão de artefactos
- Scanner OCR
- Builder de Equipas
- Otimizador Inteligente
- Assistente IA
- Relatórios Avançados
- Dashboard em tempo real
